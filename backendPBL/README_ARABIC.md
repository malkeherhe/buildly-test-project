# نظام إدارة المسارات التعليمية

## 🚀 بدء سريع

### تشغيل الباك إند

**الطريقة السريعة (Windows):**
```bash
# انقر نقراً مزدوجاً على الملف:
start_backend.bat
```

**أو يدوياً:**
```powershell
# 1. تفعيل venv
.\venv\Scripts\Activate.ps1

# 2. الانتقال لمجلد المشروع
cd projectBPL

# 3. تشغيل السيرفر
python manage.py runserver
```

الباك إند يعمل على: `http://localhost:8000`

---

### تشغيل الفرونت إند

```powershell
# 1. الانتقال لمجلد الفرونت إند
cd frontend

# 2. تثبيت الحزم (للمرة الأولى فقط)
npm install

# 3. تشغيل السيرفر
npm run dev
```

الفرونت إند يعمل على: `http://localhost:3000`

## ❓ أسئلة شائعة

### هل أحتاج venv منفصل؟
**نعم!** venv مهم لعزل الحزم. لديك venv موجود بالفعل في المجلد الرئيسي.

### كيف أعرف أن الباك إند يعمل؟
افتح المتصفح على `http://localhost:8000` - إذا رأيت صفحة Django أو رسالة خطأ، الباك إند يعمل!

### ما هي المنافذ المستخدمة؟
- الباك إند: `8000`
- الفرونت إند: `3000`
